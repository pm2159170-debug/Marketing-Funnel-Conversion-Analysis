# Marketing Funnel & Conversion Performance Analysis
**Future Interns — Data Science & Analytics Task 3 (2026)**

## Project Overview
Analyzed a marketing funnel dataset to identify drop-off points,
compare channel performance, and recommend conversion improvements.

## Tools Used
- Python (pandas, matplotlib, numpy)

## Dataset
50 records across 6 channels (Email, Organic Search, Paid Search,
Social Media, Referral, LinkedIn) covering visitors, leads, MQLs,
SQLs, opportunities, customers, cost, and revenue.

## Key Findings
- Overall conversion rate: 1.91% (38,580 visitors -> 736 customers)
- Biggest drop-off: Visitors -> Leads (84.1% lost)
- Best overall conversion: Referral (4.03%)
- Highest ROI channel: Email (3515%)
- Worst performing channel: Social Media (0.64% conversion, lowest ROI)
- Overall marketing ROI: 627.5%

## Recommendations
1. Scale up Email and Referral campaigns
2. Re-evaluate Social Media strategy
3. Improve Visitor-to-Lead conversion with better landing pages
4. Reduce Paid Search budget or renegotiate CPC rates
5. Launch referral incentive programs
